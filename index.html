Dice Game

The Dice Game I made was another game I made within Unity, it was a snakes and ladders game. However, the controller was made by me using an arduino micro-controller. The micro-controller was set up as a physical dice that could detect when it was rolled. This allowed for users to use the physical dice within Unity to play boardgames.
